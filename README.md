# Otakudesu - Anime Streaming Website

A modern anime streaming website built with PHP and MySQL, featuring a clean and responsive design.

## Features

- **Homepage** with ongoing and completed anime sections
- **Anime catalog** with alphabetical browsing
- **Search functionality** with advanced filtering
- **Genre-based browsing** with categorized lists
- **Anime details** with episode listings and information
- **Video streaming** with multiple quality options
- **Responsive design** optimized for all devices

## Tech Stack

- **Backend**: PHP 8.1+
- **Database**: MySQL 8.0
- **Frontend**: Bootstrap 5, Custom CSS
- **Icons**: Font Awesome 6.4.0
- **Video Player**: Video.js

## Installation

1. Clone the repository
```bash
git clone https://github.com/ibrayoga0/Otakudesu-script.git
```

2. Configure your web server to point to the project directory

3. Import the database schema (if available)

4. Update database configuration in `config.php`

## Project Structure

```
├── assets/
│   ├── css/           # Stylesheets
│   ├── js/            # JavaScript files
│   └── images/        # Image assets
├── includes/          # PHP includes (header, footer, sidebar)
├── *.php             # Main application files
└── config.php        # Database configuration
```

## Pages

- **Homepage** - Featured ongoing and completed anime
- **Anime List** - Alphabetical anime catalog  
- **Ongoing Anime** - Currently airing series
- **Complete Anime** - Finished series
- **Genre List** - Browse by categories
- **Search** - Find anime by title
- **Anime Detail** - Series information and episodes
- **Episode** - Video streaming page
